关于hexo

上传问题

经常遇到time out

决定更换一下repository

由原来的 HTTPS链接 更改为 SSH链接

  1. 重新生成ssh：

    运行以下命令

    ssh-keygen -t rsa -C "你的github邮箱"

  2. 在GitHub中配置SSH：

    首先在GitHub中进入setting

    在左边的侧边栏找到 SSH and GPG keys

    在SSH keys 的栏目 上 有一个 New SSH key,点击

    Title 无所谓，重要的是把在C:/user/.ssh/id_rsa.pub文件里面的内容粘贴到key里面

  3. 验证是否成功

    我这里显示：kex_exchange_identification: Connection closed by remote host

    代表链接失败了

    但是我重启了一下电脑就又成功了，可能是我电脑有问题吧。

  4. 最后修改hexo配置文件.config.yml

    在GitHub仓库中找到你的ssh

    解决！
