在您的cent OS上运行一辆小火车

发表于2022-09-17|更新于2023-03-20|Linux
|字数总计:166|阅读时长:1分钟|阅读量:|评论数:

sl 命令，可以让Linux的桌面上出现一辆小火车

但是使用前需要先安装此命令，安装也很简单

以cent OS 7为例，来看一下安装方法:

一 首先切换到root用户

在命令行窗口输入 su root

二 开始安装

正常流程

  1. 下载yum源

    wget http://mirror.centos.org/centos/7/extras/x86_64/Packages/epel-release-7-11.noarch.rpm

  2. 安装yum源

    yum install -y epel-release-7-11.noarch.rpm

  3. 安装 sl

    yum install -y sl

  4. 运行

    直接输入 sl 便可以运行啦

遇到问题

在安装中，你可能会收到提示

另一个应用程序是：PackageKit

只需要

rm -f /var/run/yum.pid
