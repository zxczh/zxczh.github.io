Typora的使用

模式切换

源代码模式 Ctrl + /

文字处理

一级标题 - 六级标题

ctrl + （1 -6）

降低标题级别 Ctrl + -

提高标题级别 Ctrl + =

list列表处理

降级快捷键 Tab

升级快捷键 Shift + Tab

选中整行 Ctrl + L

加粗 Ctrl + B

引用 Ctrl + Shift + Q

字体倾斜 Ctrl + i

高亮 ==高亮展示== 记得要在偏好设置中的拓展语法中打开才能使用

Ctrl + Shift + L 显示/隐藏侧边栏

下划线 Ctrl + U 这是下划线实例

删除线 Alt + Shift +5 这是删除线实例

无序列表 Ctrl + Shift + ]

  • 无序列表实例
    • Tab键进入子序列
      • 同样是Tab键
  • ENTER 返回列表上级

表格

ctrl + T

分割线

三个或以上 - 或者 * 并回车出现分割线

插入本地图片 直接拖入图片

插入网络图片

快捷键 CTRL + shift + i

插入链接 CTRL + K 链接实例

代码块 CTRL + shift + K

关于数学公式

$$并且回车 以下是公式示例
$$
x^2 y_1 \frac{1}{2} \cdots \sqrt{2} \vec{a} \int{x}dx
\int_{1}^2{x}dx
$$

  • x^2
  • y_1
  • \frac{1}{2}
  • \cdots \sqrt{2}
  • \vec{a}
  • \int{x}dx
  • \int_{1}^2{x}dx

此外还有大部分都支持，用得着再查吧
